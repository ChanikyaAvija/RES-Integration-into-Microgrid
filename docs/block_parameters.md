# Block Parameters

A machine-parsable dump of block parameters is included in `model/blocks_full.json`.

For quick reference, open `model/blocks_summary.csv` (CSV table) which lists blocks and a short parameter summary.

If you want a human-readable table extracted for a particular subsystem (e.g., 'Boost Converter' or 'Bidirectional DC/DC Converter'), tell me which subsystem and I'll prepare a Markdown table listing the most relevant parameters (gains, time constants, capacitor/inductor values, switching frequencies, etc.).
